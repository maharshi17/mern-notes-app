# MERN Notes App 📝

A full-stack Notes application built with:

- MongoDB
- Express.js
- React (with Vite)
- Node.js
- TailwindCSS for styling

## Features

✅ Create, Read, Update, Delete Notes  
✅ Responsive UI with Tailwind  
✅ Fully RESTful API backend  
✅ Monorepo structure (client + server)

## Setup

## Backend

```bash
cd server
npm install
npm run dev
```

### FRONTEND
